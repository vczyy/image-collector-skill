Image Collector 技能使用说明书

核心功能： 这是一个自动化的网页内容采集工具。它能智能识别网页中的文章和高清图片，提供交互式选择界面，并自动完成下载、去重和分类整理。

1. 功能亮点

  • 智能检索：自动识别网页中的文章链接和图片画廊。
  • 交互选择：不用盲目下载全站，您可以勾选自己感兴趣的内容。
  • 高清优先：自动解析网页代码，找到图片的最大分辨率版本下载。
  • 智能去重：通过计算图片内容的哈希值（SHA-256），确保不会重复下载相同的图片。
  • 文章存档：将网页文章转换为干净、无广告的 HTML 格式保存。
  • 自动分类：下载内容按 域名/日期 自动归档。

2. 安装位置

技能已安装至：E:\OpenCode\skill image-collector

3. 如何使用

方式一：交互模式（推荐）

直接运行命令，然后根据提示输入网址：

/image-collector

或

npm start

方式二：直接指定网址

/image-collector https://www.example.com/gallery

4. 使用流程演示

  1. 启动：运行命令。
  2. 输入：粘贴您想采集的网址。
  3. 分析：工具会自动打开浏览器（无头模式）扫描页面。
  4. 选择：终端会列出发现的文章或图片集，使用 空格键 勾选，回车键 确认。
  5. 下载：工具自动处理下载、查重和保存。
  6. 查看：文件保存在 E:\OpenCode\skill image-collector\downloads 目录下。

5. 常见问题

Q: 下载失败提示 Puppeteer 错误？
A: 这通常是因为 Chrome 浏览器未正确安装。您可以尝试手动运行 npm install 或设置环境变量跳过下载。

Q: 图片为什么没有下载？
A: 可能是因为触发了去重机制（该图片已存在），或者图片链接受防盗链保护。
